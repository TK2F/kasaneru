# Kasaneru 操作説明書（v5）

## はじめに

OBS の **カスタムブラウザドック**（`dock.html`）と **ブラウザソース**（`overlay.html`）の 2 画面で動作します。詳細なインストールは [README.md](../README.md) を参照してください。

**解像度**: ドックの数値はすべて **1920×1080 の論理座標**です。Browser Source を **1280×720** などにしても、オーバーレイ内で **均等スケール**されます。OBS のキャンバス・出力の組み合わせは [BROADCAST-SETUP.md](./BROADCAST-SETUP.md)、仕様の端的な定義は [SPEC.md](./SPEC.md) を参照してください。

---

## 画面構成

| タブ | 役割 |
|------|------|
| **チャプター** | メインの章テロップ。LIVE / HIDE、前後移動、編集・並べ替え。 |
| **タイトル** | TitleCard（ラベル付きカード）。SHOW / HIDE。 |
| **画像** | 画像オーバーレイ。SHOW / HIDE。 |
| **グループ** | 上記 3 つで共有するグループ（色・名前）。 |
| **設定** | テーマ、アクセント、レイヤー順、オフセット、ガイド、CSV など。 |

### Live / Offline（右上トグル）

画面右上（言語切替の右隣）に **`LIVE / OFFLINE`** トグルがあります。

- **LIVE**: 各セクションの状態に応じて配信へ表示されます（チャプターの「表示 ▶」、TitleCard/画像の「SHOW ▶」など）。
- **OFFLINE**: **チャプター / TitleCard / 画像を強制的にすべて非表示**にします。各セクションの状態は保持され、LIVE に戻すと「表示になっているものだけ」復帰します。

### 画像パスと `images/` フォルダ（推奨運用）

- **相対パス推奨**: `overlay.html` と同じフォルダ直下の **`images/`** に素材を置き、登録は **`images/ファイル名.png`** のように書きます（ドックの「画像」タブのプルダウン・手入力・CSV いずれも利用可）。
- **プルダウン（`images/` から選択）**: ブラウザはローカルフォルダを自動列挙できないため、候補一覧は **`images/manifest.json`** を読み込みます。画像を追加・削除したら、プロジェクト直下で次を実行して更新してください。  
  `powershell -ExecutionPolicy Bypass -File .\tools\generate-image-manifest.ps1`
- 環境によって `file:///` 上の XHR が制限されるため、生成スクリプトは **`images/manifest.js`** も出力します（ドックは JS→JSON の順で読み込みを試みます）。
- **フルパス**: 従来どおり **`C:/Pictures/...`** のような絶対パスも登録できます。「登録形式」で **フルパス / file URL** を選ぶと、`file:///` で開いたドックから **file URL** 形式で埋める場合があります（環境によりローカル絶対パスへ変換できないときのフォールバック）。
- **対応拡張子（manifest に載る）**: png / jpg / jpeg / webp / svg / gif / bmp（大文字小文字は問わず）。

### Chapters / TitleCard / Images タブの 3領域

各タブは次の 3 領域に固定で分かれます。

1. **追加した項目**（リスト）
2. **追加 / 編集**（フォーム）
3. **表示 / 操作**（LIVE / SHOW / HIDE / 前後 / ロック）

運用ルール:

- スクロールは原則 **追加した項目（リスト）** のみ
- **追加 / 編集** と **表示 / 操作** は固定領域で、誤スクロールによる操作ミスを防止
- 見出しで視覚的に区切り、初見でも「どこを見るか」が分かる構成

## 配信ロック（編集ロック）

各タブ（チャプター / タイトル / 画像）のコントロール下部 **「編集ロック…」** は **共通スイッチ**です（どのタブで押しても同じ状態になります）。オンにすると:

- **できないこと**: 追加・複製・編集・削除・ドラッグ並べ替え（CSV 取り込みも不可）
- **できること**: 前後移動、表示 / 非表示（SHOW / HIDE）など **表示の切り替え**

配信本番で誤操作を防ぐ用途。**ロック解除**で元に戻ります。

補足:

- **状態表示**: タブ左の丸ポチ（チャプター/タイトル/画像が**表示中**のとき緑系で点灯）。編集ロックは各タブの **編集ロック…** ボタン文言が **ロック解除** になっているかで確認できます。

---

## 元に戻す（Undo）

フッターの **「元に戻す」**（英語 UI では **Undo**）または **Ctrl+Z** で、直前の編集操作を取り消します。チャプター・TitleCard・画像・グループ変更など、直前スナップショット単位で復元されます。

---

日本語 / English は**画面上部右端**のスイッチで切り替え（設定は保存されます）。

---

## レイアウトガイド（Browser Source 上の点線）

**設定 → グローバルオフセット**付近の **ガイド** で動作を選びます。

| モード | 挙動 |
|--------|------|
| **自動** | チャプター表示中、TitleCard 表示中、画像表示中のいずれかのときは **ガイド非表示**。設定タブを開いているとき、または TitleCard/画像の **オフセット編集パネルを開いていて最小化していない** ときに表示。 |
| **常に表示** | 上記に関係なくガイドを表示（配信中もソース上に残るので注意）。 |
| **常に非表示** | 常にクリア。 |

**グリッド**はガイド表示中に三分割線を重ねます。

オフセット入力中にガイドがちらつく場合は、未送信のプレビューをキャンセルするよう調整済みです。配信開始・終了・SHOW/HIDE でもガイド状態が更新されます。

## テーマと CSS

- **Style（テーマ）**: `themes/` 内の CSS（Default / Signal / Airy / Pulse）。
- **Extra CSS / User styles**: `overlay.html` からの相対パスで追加スタイルを読み込み。
- **Dock の強調色**: 設定の外観で **ドック UI 用アクセント**を指定可能。**ストリームに合わせる**でオーバーレイ用アクセントと一致させられます。

### TitleCard の「ラベル」と「カード背景」

- **ラベル（文言）**: TitleCard のカード上に付く **小さなタグ**（例: `Q&A`）。**文言は TitleCard の追加 / 編集でカードごとに設定**（**任意 1 行**、空欄でラベルなし）。**チャプターのバッジ（CH.1）とは別物**です。
- **ラベル（見た目）**: 色・字体・**ラベル背景**（カード色同期／アクセント同期）・**面の不透過**は **設定 → タイトルカードのスタイル**で一括指定します。
- **バッジ/ラベル色**: CH バッジと TitleCard ラベルの“色ルール”を **アクセント統一 / コンテキスト同期**で切り替えられます。
- **TC カード背景**: Dock のカラーピッカー（色＋透明度）。**クリア**で **チャプターのカード背景**にフォールバックします。

補足:

- **アクセント統一**を選ぶと、ラベル背景の同期先も実質アクセント扱いになります。
- **ラベル＝バッジ字体**: 既定 ON。章番号バッジと同じサイズ・太字・斜体に揃えます。OFF でラベル用フォントを個別指定できます。

### チャプター配信の起動時

- ドックを開き直した直後、チャプターは **常に非表示（オフエア）** から始まります。オーバーレイに出す場合は **表示 ▶** を押してください（前回セッションで ON のままにはしません）。

## データ（CSV とリセット）

**設定 → Data** で次を使い分けられます。

| 操作 | 効果 |
|------|------|
| **CSV 種別 + 書き出し / 読み込み** | **チャプター** / **タイトルカード** / **画像** / **Apps** / **UI・見た目** / **設定パック(JSON)**。列定義と例は [CSV-TEMPLATES.md](./CSV-TEMPLATES.md) と `templates/csv/`。リスト系取り込みは**そのリストをすべて置き換え**。 |
| **見た目・配置リセット** | テーマ・フォント・位置・TitleCard スタイル・ガイド等を初期化。**登録したチャプター / タイトルカード / 画像は残ります。** Dock の表示言語も維持されます。 |
| **登録データのみ消去** | CH・グループ・TC・画像の**登録だけ**クリア。**設定（見た目）は残ります。** 編集ロック ON 中は実行できません。 |
| **すべてリセット** | 登録データと見た目設定の**両方**を初期化（Dock 言語は JA に戻ります）。 |

自作テーマの手順は [CUSTOM-CSS-GUIDE.md](../CUSTOM-CSS-GUIDE.md) を参照。

### CSV と JSON の使い分け

- **CSV**: チャプター台本、タイトルカード文言、画像差し替え、Apps一覧の編集に向く。
- **UI 設定 CSV**: 色/フォント/配置などをスプレッドシートで差分管理したいときに向く。
- **設定パック(JSON)**: 本番環境の完全バックアップ、別PCへの移行、配信イベントごとの「凍結版」保存に向く。

## 外部制御（Stream Deck / WebSocket）

`dock.html` は任意で外部 WebSocket 制御を受けられます。  
起動URLに以下を付与します。

- `remoteWs=ws://127.0.0.1:9099`
- `remoteEnabled=true`
- `remoteToken=任意トークン`（省略可）
- `remoteReconnectMs=2500`（省略可、500–60000）

受信JSON例:

```json
{ "cmd": "control", "action": "chapter_next", "token": "abc", "requestId": "r1" }
```

主な `action`:
- `chapter_live`, `chapter_hide`, `chapter_prev`, `chapter_next`
- `titlecard_show`, `titlecard_hide`, `titlecard_prev`, `titlecard_next`
- `image_show`, `image_hide`, `image_prev`, `image_next`
- `app_show_all`, `app_hide_all`, `app_prev`, `app_next`
- `master_on`, `master_off`, `sync`
- `set_tab`（`tab: "chapters"|"titlecard"|"images"|"groups"|"apps"|"settings"`）
- `chapter_select`（`index: 0..n-1`）

`requestId` がある場合、dock は ACK を返します:

```json
{ "cmd": "ack", "requestId": "r1", "ok": true, "action": "chapter_next" }
```

|------|--------|----------------|-----|

## トラブルシューティング

| 現象 | 確認 |
|------|------|
| ドックとソースで連携しない | 両方 `file:///` で**同じパス階層**か。ソースの **「ローカルファイル」はオフ**。 |
| 画像が出ない | パスは `overlay.html` からの**相対パス**、または **絶対パス（例: `C:/...`）** を画像タブで直接指定。バックスラッシュは保存時に `/` に揃われます。日本語・スペースを含むパスはオーバーレイ側で解決時にエンコードされます。ファイル実在と Browser Source のローカルファイル設定を確認。 |
| 画像プルダウンが空／読めない | `images/` に画像を置き、`tools/generate-image-manifest.ps1` で **`images/manifest.json`** を生成・更新する。相対パスは **`images/ファイル名`** で指定。 |
| ガイドが消えない | ガイドが **常に表示** になっていないか。設定タブを開いたままか。 |
| ガイドが出ない | **自動**で配信中でないか。**常に非表示**でないか。TC/画像の編集パネルが閉じているか。 |
