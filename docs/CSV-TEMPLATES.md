# CSV インポート / エクスポート（チャプター・タイトルカード・画像・UI 設定）

> 2026-04 更新: `設定パック(JSON)` を追加。  
> 日常運用の「素材差し替え・台本更新」は CSV、環境移行や完全バックアップは JSON を推奨。

## ドックでの操作

1. **設定**タブ → **Data** セクションを開く。
2. **CSV 種別**で対象を選ぶ（チャプター / タイトルカード / 画像 / **UI・見た目** / **設定パック(JSON)**）。
3. **CSV 書き出し**で現在のデータをダウンロード。**CSV 読み込み**でファイルを選択。
4. **リスト系**（チャプター・TC・画像・Apps）: 既存の同種リストは**置き換え**。編集ロックがオンのときは読み込みできません。
5. **UI・見た目**（配置・フォント・テーマ等）: 1 行目が `key,value` のテーブル。**編集ロック中でも**読み込み可能。左列のキーはドックの書き出しファイルを基準にすると安全です。一部のキーだけのファイルでも、そのキーのみ反映されます。
6. **設定パック(JSON)**: `lt_ch_v5` 相当の状態を丸ごと保存・復元。OBS PC 間移行や配信レイアウトのスナップショット保存向け。

## 列定義

### チャプター（`kasaneru-chapters.csv`）

| 列 | 必須 | 説明 |
|----|------|------|
| group | 任意 | グループ名。存在しなければ作成される |
| title | **必須** | タイトル |
| subtitle | 任意 | サブタイトル |

### タイトルカード（`kasaneru-titlecards.csv`）

| 列 | 必須 | 説明 |
|----|------|------|
| group | 任意 | グループ名 |
| title | **必須** | カードタイトル |
| label | 任意 | 単一ラベル（空＝ラベルなし） |
| color | 任意 | `#RRGGBB`。空なら `#e8002d` |
| offsetX / offsetY | 任意 | 整数（ピクセル） |

### 画像（`kasaneru-images.csv`）

| 列 | 必須 | 説明 |
|----|------|------|
| group | 任意 | グループ名 |
| name | 任意 | 一覧表示名。空ならファイル名を使用 |
| path | **必須** | 相対 or 絶対パス |
| scale | 任意 | 百分率。省略時 100 |
| offsetX / offsetY | 任意 | 整数 |
| fit | 任意 | `default` / `fit` / `overflow`（または true/false 相当） |

### アプリ（`kasaneru-apps.csv`）

| 列 | 必須 | 説明 |
|----|------|------|
| name | **必須** | アプリ表示名 |
| url | **必須** | 表示 URL（`overlay.html` からの相対パス。例: `apps/timer.html`） |
| ctrlUrl | 任意 | コントロール URL（空の場合、`url?ctrl=1&dock=1` にフォールバック） |
| position | 任意 | 9 点アンカー（`tl` `tc` `tr` `cl` `center` `cr` `bl` `bc` `br` `full`）。省略時 `tr` |
| offsetX / offsetY | 任意 | 整数（ピクセル）。省略時 0 |
| width / height | 任意 | 整数（ピクセル）。省略時 480 / 320 |
| zLayer | 任意 | 重なり順。省略時 50 |
| opacity | 任意 | 0–100。省略時 100 |
| animMode | 任意 | `default` / `fade` / `none` |
| borderRadius | 任意 | 整数（px）。省略時 0 |
| group | 任意 | グループ名 |
| scale | 任意 | 百分率。省略時 100 |
| interactive | 任意 | `1` = クリック可 / `0` = 表示のみ。省略時 0 |
| themeSync | 任意 | `1` = テーマ同期 / `0` = しない。省略時 0 |

### UI・見た目（`kasaneru-ui-settings.csv`）

| 列 | 必須 | 説明 |
|----|------|------|
| key | **必須** | 設定識別子（例: `theme`, `accentColor`, `font`, `layerOrder` …） |
| value | **必須** | 文字列。オブジェクト・配列は **JSON を 1 セル**に入れる（書き出しと同じ形式） |

- 先頭行は必ず `key,value`。
- `font`, `dockUi`, `layerOrder`, `userStylePaths`, `labelSettings`, `titleCardFont`, `collapsedGroups` などは JSON。ブールは `true` / `false`。
- 未知のキーは無視されます。値の形式が不正なキーはスキップされます。
- `masterLive` は右上の **LIVE / OFFLINE** トグル状態です（`true`/`false`）。OFFLINE を取り込んだ場合、配信へは強制的に何も出なくなります。

## Google スプレッドシートでの作り方（おすすめ）

1. 1 行目に上記の **ヘッダー行**をそのまま入力する（テンプレは `templates/csv/*.template.csv`）。
2. 2 行目以降にデータを入力。グループ名はチャプター・TC・画像で**同じ表記**にすると同じグループ ID に紐づきます。
3. **ファイル → ダウンロード → カンマ区切り形式 (.csv)** で保存。
4. ドックで種別を合わせて **CSV 読み込み**。

注意: スプレッドシートの「数式の結果」で書き出す。結合セルは避ける。

## トラブルシュート

- **文字化け**: UTF-8 で保存されているか確認する。
- **取り込み 0 件**: 必須列が空の行はスキップされる。ヘッダー行を 1 行目に置く。
- **画像が表示されない**: `docs/OPERATION-GUIDE.md` の画像パス節を参照。
